## Installation

#### Clone the repository

```
$ git clone https://github.com/serverless/enterprise.git
```

#### Install Front-End & Back-End Dependencies

```
# location - enterprise/frontend
$ npm i
```

```
# location - enterprise/backend
$ npm i
```

#### Deploy the back-end

```shell
# location - enterprise/backend
$ serverless deploy
```

#### Run the front-end

```
# location - enterprise/backend
$ npm run start
```
